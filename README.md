# BlackEagleLK

BlackEagleLK is a movie streaming and discovery website built with React and Vite. It is designed to give users a cinematic browsing experience with a polished movie detail page, searchable catalog, category pages, and built-in subtitle controls.

## What the site is meant to be

This site is meant to be a dark, premium movie platform where users can:

- Browse trending, popular, top-rated, and category-based movies
- Open a movie detail page with a cinematic hero section
- Watch movies in an embedded player
- Use subtitle controls and styling options
- View cast details and related movie recommendations
- Search for titles quickly from the navbar

## Main Sections

- Home page: featured movie rows and category shortcuts
- Movie detail page: hero banner, player, subtitles, cast, and related movies
- Category pages: language and movie category listings
- Search page: movie search results

## Tech Stack

- React
- Vite
- TMDB API
- React Router

## Notes

This project is focused on a clean, dark streaming-style UI with a premium feel similar to modern movie platforms.
